Class Selectors

1- We are selecting and manipulating in pure JS which uses getElementsByClassName("class-name")[#] to find the div class (NOTE: no . is required with this selector). You may notice [0] attached to the selector and wonder, "Why is this here and not for getElementById()?". HTML only allows one ID per tag whereas you can have multiple classes. So the getElementById() method returns a precise value whereas getElementsByClassName() method returns a JS array object of all tags that contain the class. In our very simplified example, we know there is only one instance of js02class so we can just call position 0 in our array object and get the result we want.
document.getElementsByClassName("js02class")[0].addEventListener("click", function() { document.getElementsByClassName("stronghold")[0].classList.toggle("red"); });

2- This does the EXACT same thing as 1 using pure JS and the same methods but using the petworth function and declaring the selectors as variables. The function encapsulates this into a mini-program and properly scopes the variables to limit themselves to only this function. This is a best-practice way to write JS. function petworth ( ButtonID, SelectorID ) { var petworthButton = document.getElementsByClassName(ButtonID)[0]; var petworthSelector = document.getElementsByClassName(SelectorID)[0]; petworthButton.addEventListener("click", function(){ petworthSelector.classList.toggle("red"); }); } petworth( "js03class" , "petworth" );

3- This codeblock maintains the function variable and setup from 2 but uses querySelector instead of getElementsByClassName() for selecting the class. Your not-so-humble prof thinks querySelector() is much better than getElementsByClassName() because the selector syntax is uniform between IDs and classes, similar to jQuery. That's why you have to add . for class queries and # for ID queries when you pass the declarations into the function. function dupont ( ButtonID, SelectorID ) { var dupontButton = document.querySelector(ButtonID); var dupontSelector = document.querySelector(SelectorID); dupontButton.addEventListener("click", function(){ dupontSelector.classList.toggle("red"); }); } dupont( ".js04class" , ".dupont" );
